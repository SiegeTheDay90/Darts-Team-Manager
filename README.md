# Darts-Team-Manager

## MVPs

### User-Auth
<ul>
  <li>Managers can Log In to create/edit rosters</li>
</ul>

### Player-View
<ul>
  <li>Players can Log In to view their team(s)</li>
  <li>Can "check-in" to reserve a spot for game-day</li>
  <li>Can list "preffered games" and time slots.</li>
</ul>
